# DevStreak

Full-stack app to track daily development streak.

## Tech Stack

- Frontend: React (Vite)
- Backend: Node.js + Express

## Setup

### Backend

cd backend
npm install
node server.js

### Frontend

cd frontend
npm install
npm run dev

## Features

- Backend API (/api/message)
- Frontend connected to backend
- CORS configured
